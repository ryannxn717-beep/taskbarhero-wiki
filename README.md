# Task Bar Hero Wiki

An independent community wiki for Task Bar Hero, with game data, equipment, heroes, stages, skills, runes, builds, guides and interactive tools.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run check
npm run build
```

The project is built with SvelteKit and the static adapter.

Task Bar Hero names and artwork belong to their respective rights holders. This repository is an independent fan project and is not affiliated with the game developer.
